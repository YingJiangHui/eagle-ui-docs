# 选项

# 1. 图标

# 1.1 设置svg图标

使用 icon 属性在Button中嵌入svg图标。

参数：setting、info、error、download、zan、left、right、loading

# 1.2 设置图标位置

使用 iconPosition 修改图标的位置，默认为 left

参数：left、right

# 2. 加载动画

使用 loading 设置是否出现加载动画。

参数：Boolean

# 3. 禁用

使用 disabled 属性设置按钮是否被禁用

参数：Boolean
